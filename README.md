# C_Exam 2018

## How to run
1. From root folder in the project, run the following commands:
    - `make`
    - `./bin/main`

### Aditional make commands
- `make clean`
  - cleans up autogenerated files
- `make remake`
  - cleans up autogenerated files and builds the program
